# Symplectic Geometry

## Hamiltonian Systems over the Cotangent Bundle

Recall many systems over configuration space given by some smooth manifold \\(M\\) may be modeled by a *Hamiltonian system*, which considers phase space \\(T^\*M\\) together with a *Hamiltonian* \\(H: T^\*M \to \mathbb{R}\\). Then, a solution \\((x(t), \xi(t))\\) over \\(T^\*M\\) to this Hamiltonian system is defined by the requirement it satisfies *Hamilton's equations*, which is given by
\\[
    \begin{align}
        \dot{x}^i &= \frac{\partial H}{\partial \xi^i}\\\\\
        \dot{\xi}^i &= -\frac{\partial H}{\partial x^i}.
    \end{align}
\\]
in some choice of *natural coordinates* \\((x^1, \cdots, x^n, \xi^1, \cdots, \xi^n)\\). Equivalently, solutions are integral curves of the flow generated by the vector field
\\[
    X\_H = \sum\_i \frac{\partial H}{\partial \xi^i} \partial\_{x^i} - \frac{\partial H}{\partial x^i} \partial\_{x^i}.
\\]
Further equivalently, if we define the ***canonical symplectic form***
\\[
    \omega := \sum\_i dx^i \wedge d\xi\_i,
\\]
then \\(X\_H\\) is nicely characterized by the coordinate-free condition
\\[
    i\_{X\_H} \omega = dH.
    \label{eq:hamilton}
    \tag{H}
\\]
Indeed, to see \\(X\_H\\) satisfies the above equation, compute
\\[
    \begin{align}
        i\_{X\_H} \omega
        = i\_{X\_H} \left( \sum\_i dx^i \wedge d\xi^i \right)
        = \sum\_i dx^i(X\_H) d\xi^i - d\xi^i(X\_H) dx^i
        = \sum\_i \frac{\partial H}{\partial \xi^i} d\xi^i + \frac{\partial H}{\partial x^i} dx^i 
        = dH.
    \end{align}
\\]
Furthermore, note that by the middle expression in the above equality chain, if \\(i\_{X\_H} \omega = 0\\), then
\\[
    \sum\_i dx^i(X\_H) d\xi^i - d\xi^i(X\_H) dx^i = 0 \implies X\_H = 0.
\\]
That is, the linear map \\(X\_H \mapsto i\_{X\_H} \omega\\) is an isomorphism and therefore \eqref{eq:hamilton} uniquely characterizes \\(X\_H\\). In general, we say a 2-form \\(\omega\\) over a manifold \\(N\\) is ***non-degenerate*** if \\(v \mapsto i\_v \omega\\) is an isomorphism for all \\(p \in N\\). Additionally observe the canonical symplectic form can be rewritten
\\[
    \omega\_{(x, \xi)}
    = \sum\_i dx\_i \wedge d\xi^i 
    = \sum\_i d(\xi\_i dx^i) 
    = d(\pi^\*\_{(x, \xi)}\xi)
    = d\tau\_{(x, \xi)}
\\]
for ***tautological 1-form*** \\(\tau\_{(x, \xi)} := d\pi^\*\_{(x, \xi)}\xi\\). To summarize, given the cotangent bundle \\(T^*M\\) with the canonical symplectic form \\(\omega = d\tau\\), Hamilton's equations for a Hamiltonian \\(H: T^\*M \to \mathbb{R}\\) are encoded in the vector field \\(X\_H\\) characterized by \\(i\_{X\_H} \omega = dH\\). Note the canonical symplectic form is non-degenerate and exact and therefore closed. Thus the definition given in the next section is an immediate generalization of this.

## Symplectic Manifolds

A ***symplectic manifold*** is a \\(2n\\)-dimensional smooth manifold \\(N\\) together with a closed, non-degenerate \\(2\\)-form \\(\omega\\) called the ***symplectic form***. A ***Hamiltonian*** is a smooth function \\(H: T^\*M \to \mathbb{R}\\), and a ***Hamiltonian system*** is a symplectic manifold \\((N, \omega)\\) together with a Hamiltonian \\(H\\). This induces a ***Hamiltonian flow***, which is generated by the ***Hamiltonian vector field*** \\(X\_H\\) characterized by
\\[
    i\_{X\_H} \omega = dH.
\\]
Now, some comments on this definition. Recall the canonical symplectic form over \\(T^\*M\\) is exact while this general definition weakens the condition to only require a symplectic form to be closed. We will see later (in the *Darboux Theorem*) that a symplectic manifold is still locally equivalent to \\(T^*M\\) with canonical symplectic form. Furthermore, we still require the symplectic form to be closed because this provides crucial structure. For example, given a Hamiltonian system \\((M, \omega, H)\\), notice the symplectic form is invariant under the Hamiltonian flow only because \\(\omega\\) is closed:
\\[
    \mathcal{L}\_{X\_H} \omega = d(i\_{X\_H} \omega) + i\_{X\_H} d\omega = d^2 H + i\_{X\_H} d\omega = 0.
\\]
