## Linear Independence

#### Definition
A set of vectors $v_1, v_2, \cdots, v_n$ are said to be *linearly independent* if the only scalar solution that allows $\sum_{i=0}^n \alpha_i v^i = 0$ is $\alpha_1 = \alpha_2 = \cdots = \alpha_n = 0$.
